# Anki 费曼学习法插件配置说明

## OpenAI 配置
- `api_key`: OpenAI API密钥
- `model`: 使用的模型名称，默认为 gpt-3.5-turbo
- `max_tokens`: 单次请求最大token数
- `temperature`: 生成文本的创造性程度 (0-1)
- `request_timeout`: API请求超时时间（秒）

## 界面设置
- `theme`: 界面主题，可选 "default", "dark"
- `language`: 界面语言，目前支持 "zh_CN"
- `window_size`: 窗口大小设置
  - `width`: 宽度（像素）
  - `height`: 高度（像素）

## 牌组设置
- `default_deck`: 默认保存牌组
- `error_deck`: 错题牌组名称
- `tags`: 默认添加的标签列表
