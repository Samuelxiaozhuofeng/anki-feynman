# Anki 费曼学习法插件

这是一个基于费曼学习法的Anki插件，通过AI辅助生成问题和评估答案，帮助用户更好地理解和记忆知识点。

## 功能特点

1. **费曼学习法实现**：通过AI辅助，将复杂概念分解成简单问题，帮助理解和记忆
2. **多种题型支持**：生成选择题、问答题和知识卡，满足不同学习需求
3. **AI评估反馈**：对回答进行智能评估，提供针对性反馈
4. **知识卡片创建**：将复杂内容提炼为易于理解的知识卡片，支持填空卡转换
5. **语言学习助手**：提供语言句型替换练习，帮助掌握语言模式（新增！）
6. **支持Anki集成**：无缝整合进Anki复习流程

## 安装要求

- Anki 2.1.50 或更高版本
- Python 3.9 或更高版本
- OpenAI API密钥

## 安装方法

1. 在Anki中点击"工具" -> "附加组件"
2. 点击"获取插件"
3. 输入插件代码：[待添加]

## 配置说明

首次使用时需要配置OpenAI API密钥：

1. 点击"工具" -> "费曼学习设置"
2. 在设置窗口中输入API密钥
3. 根据需要调整其他设置

详细配置说明请参考 `config.md`。

## 使用方法

1. 选择或创建笔记
2. 点击"费曼学习"按钮
3. 输入要学习的文本
4. 点击"生成问题"
5. 回答生成的问题
6. 查看AI评估反馈

## 使用指南

### 费曼学习

1. 点击Anki顶部菜单中的"费曼学习法"选项
2. 在弹出窗口中输入您想学习的内容
3. 选择目标牌组和题型（选择题/问答题/知识卡）
4. 点击"生成问题"按钮
5. 逐一回答生成的问题，获取AI评估反馈
6. 可一键将问题和答案添加至Anki牌组

### 语言学习助手（新增！）

1. 点击Anki顶部菜单中的"费曼学习法" -> "语言学习练习"
2. 选择目标语言和牌组
3. 输入想练习的句子
4. 点击"生成练习"按钮
5. 查看生成的句型分析和可替换部分
6. 浏览替换示例，可一键添加到Anki牌组进行记忆
7. 重复以上步骤，练习更多句型

特点：
- 系统会分析句子结构，识别核心句型
- 提供可替换的词汇和短语建议
- 生成符合语言水平的例句
- 包含详细的语法解释和翻译
- 一键添加到Anki牌组功能
