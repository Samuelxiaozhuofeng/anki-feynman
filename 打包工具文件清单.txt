========================================
Anki Feynman Learning 插件打包工具
文件清单
========================================

已创建的文件：

1. build_addon.py
   - 核心打包脚本
   - 功能：自动打包插件，排除敏感信息，包含所有依赖
   - 使用：python build_addon.py

2. 打包插件.bat
   - Windows 快捷打包脚本
   - 功能：一键运行打包脚本
   - 使用：双击运行

3. 安装依赖.bat
   - Windows 快捷安装依赖脚本
   - 功能：一键安装所有依赖到 vendor 目录
   - 使用：双击运行

4. 打包说明.md
   - 详细的打包说明文档
   - 包含：功能特性、配置说明、常见问题

5. README_打包工具.md
   - 完整的使用指南
   - 包含：快速开始、详细步骤、故障排除

6. 打包工具文件清单.txt
   - 本文件
   - 列出所有相关文件

========================================
使用流程：
========================================

首次使用：
1. 双击运行 "安装依赖.bat"
2. 等待依赖安装完成

每次打包：
1. 双击运行 "打包插件.bat"
2. 在 dist 目录找到生成的 .ankiaddon 文件
3. 将文件发送给其他用户

========================================
重要提示：
========================================

✅ 已自动处理：
- API keys 自动清空
- 开发文件自动排除
- 依赖库自动包含
- 临时文件自动清理

⚠️ 需要注意：
- 首次使用前必须安装依赖
- 打包前确保 vendor 目录存在
- 打包后的文件不包含用户配置
- 用户需要自行配置 API key
